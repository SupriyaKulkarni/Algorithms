Problem Description
Weighted increasing sub sequence(input will be a array of elements and and another array with corresponding weights) 
a) to find the sub sequence that is increasing and has max weights
b) same as a) but can violate the increasing sub sequence at-most 'n' times but must get max weight.

Compile the file "weighted_increasing_sequence.c" with gcc command
--gcc weighted_increasing_sequence.c

To see the output use ./a.out and then enter the input on the stdin
